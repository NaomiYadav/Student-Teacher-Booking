🚀 Demo Setup Guide

Firebase Configuration Required

To run this application, you need to configure Firebase:

  1. Create a Firebase Project:
    • Go to Firebase Console
    • Click "Create a project"
    • Follow the setup wizard
  2. Enable Services:
    • Enable Authentication (Email/Password method)
    • Create a Firestore database in test mode
    • Enable Firebase Hosting
  3. Get Configuration:
    • Go to Project Settings > General
    • Scroll down to "Your apps"
    • Click "Add app" > Web app
    • Copy the Firebase configuration object
  4. Update Configuration:
    • Open js/firebase-config.js
    • Replace the placeholder config with your Firebase config
    • Save the file
  5. Deploy or Run Locally:
    • For local development: Use a local server (Live Server extension in VS Code)
    • For production: Run firebase deploy

🔧 Development Tools

  • VS Code Extensions: Live Server, Firebase
  • Browser: Chrome with Developer Tools
  • Firebase CLI: npm install -g firebase-tools

📱 Test Accounts

After setup, create test accounts for different roles:

  • Student: student@test.com
  • Teacher: teacher@test.com
  • Admin: admin@test.com (requires manual role assignment)

🎯 Demo Features

👨‍🎓 Student Features

  • Search teachers by department
  • Book appointments with teachers
  • Send messages to teachers
  • View appointment history

👨‍🏫 Teacher Features

  • Manage appointment requests
  • Approve/cancel appointments
  • View student messages
  • Set availability schedule

🛡️ Admin Features

  • Approve teacher registrations
  • Manage all users
  • View system analytics
  • Download system logs
